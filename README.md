Pokebu
=============

Pocket reader app for iOS by RubyMotion.

## アプリについて
「あとで読む」サービス、Pocketの非公式クライアントアプリです。

Pocketに保存した未読記事を消化・アーカイブしながら、アプリ内ではてなフックマークのコメントを閲覧したり、ブックマークに追加したりできます。
